# CSR Study Design Workflow

1. Copy the relevant CSP study design text (e.g., Section 3.1 and any nearby relevant design text).
2. Run `study_design_element_extractor.md`.
3. Review the extracted elements quickly for completeness and obvious misses.
4. Run `csr_study_design_generator.md` using the extracted elements.
5. Select a level of detail: Expanded / Standard / Abbreviated.
6. Run `csr_study_design_verifier.md` against the generated section.
7. Perform final MW review and refine as needed.
